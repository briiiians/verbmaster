# VerbMaster 🎓

Aplicación interactiva para aprender verbos irregulares en inglés.

## Características

- 📚 **Modo Estudio**: Aprende los 100 verbos irregulares más comunes
- 🎯 **Quiz Interactivo**: Prueba tus conocimientos con diferentes tipos de preguntas
- 🔊 **Pronunciación**: Escucha la pronunciación de cada verbo
- 📊 **Seguimiento**: Mantiene registro de tus errores para practicar
- 🌙 **Modo Oscuro**: Interfaz cómoda para los ojos

## Tipos de Quiz

1. **Infinitivo → Pasado**: Escribe el pasado del verbo en infinitivo
2. **Pasado → Participio**: Escribe el participio del pasado
3. **Infinitivo → Participio**: Escribe el participio del infinitivo
4. **Español → Infinitivo**: Traduce del español al inglés
5. **Aleatorio**: Combina todos los tipos

## Instalación Local

```bash
npm install
npm start
```

## Build para Producción

```bash
npm run build
```
